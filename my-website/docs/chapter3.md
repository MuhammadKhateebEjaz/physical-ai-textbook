# Chapter 3: ROS 2 Fundamentals

## What is ROS 2?
ROS 2 (Robot Operating System) is a middleware framework used to build robot software systems.

## Core Concepts
- Nodes
- Topics
- Services
- Actions

## Why ROS 2 for Humanoid Robots?
- Modular architecture
- Real-time support
- Distributed communication
- Large ecosystem
